Features of Creating Static Web Pages

A website is a collection of web pages, which are simple HTML files. When you send a request to a website server, the browser receives a bunch of data in response. And there will always be HTML in it, which is indispensable.

Each web page is a separate HTML file. Every time we navigate to another route, the browser makes a new request to get the HTML file for the new page. The best way to check this is to pay attention to the loading indicator in your browser. If it shows that the site is loading, you have indeed sent a request to the server, and the browser is waiting for a response.

Usually, the design of such a site is limited to images in .jpg or .png format. If you want to diversify the website, you can do it using JavaScript animation.
